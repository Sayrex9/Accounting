# Домашнее задание: «Бухгалтерия» (лекция 1. Import. Module. Package)

Реализация структуры программы «Бухгалтерия» с использованием модулей и пакетов Python.

## Цель задания

Отработать навыки:
* создания структуры пакетов и модулей в Python;
* организации импорта функций между модулями;
* работы с модулем `datetime`;
* управления зависимостями через `requirements.txt`;
* использования конструкции `from ... import *`.

## Описание модулей

**main.py**
* основной файл для запуска программы;
* импортирует функции `calculate_salary()` и `get_employees()`;
* при запуске выводит текущую дату и время, затем вызывает импортированные функции.

**dirty_main.py**
* демонстрирует использование конструкции `from package.module import *`;
* импортирует все публичные объекты из `salary.py` и `people.py`;
* выполняет те же действия, что и `main.py`, но с другим способом импорта.

**application/salary.py**
* содержит функцию `calculate_salary()`;
* функция выводит сообщение с текущей датой о начале расчёта зарплаты.

**application/db/people.py**
* содержит функцию `get_employees()`;
* функция выводит сообщение с текущей датой о получении данных о сотрудниках.

## Использование

### Предварительные требования

* Python 3.6 или выше;
* установленный менеджер пакетов `pip`.

### Установка зависимостей

* Установите зависимости из файла `requirements.txt`:
```
pip install -r requirements.txt
```

## Запуск программы

### Основной вариант (явный импорт):

```
python main.py
```
### Дополнительный вариант (импорт *):

```
python dirty_main.py
```

## Ожидаемый вывод
* При запуске любого из скриптов вы увидите:

```
Программа «Бухгалтерия» запущена. Текущая дата и время: 02.05.2026 14:30:25
[02.05.2026] Выполняется расчет зарплаты...
[02.05.2026] Получаю данные о сотрудниках...
```
